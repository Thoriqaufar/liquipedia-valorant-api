# Liquipedia Valorant API

Inspired by https://liquipedia.net/valorant
## Tech Stack

**Backend:** Golang Echo, GORM

**Database:** MySQL


## Run Locally

Clone the project

```bash
  git clone https://github.com/thoriqaufar/liquipedia-valorant-api
```

Go to the project directory

```bash
  cd my-project
```

Start the server

```bash
  go run main.go
```
